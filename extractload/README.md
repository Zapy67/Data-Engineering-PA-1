# Data Extraction and Loading

## This Project has acquired data from the following sources:
1. TRENDS IN ELECTRICITY GENERATION (2006-07 to 2020-21) - Pakistan Bureau of Statistics - Time Series Data
2. Pakistan Electricity Generation by Solar, Solar Power Production: Monthly Electricity Generation Dataset. Kaggle Link: https://www.kaggle.com/datasets/ahmadwaleed1/pakistan-electricity-generation-by-solar
3. Pakistan Solar Radiation Dataset. Kaggle Link: https://www.kaggle.com/datasets/muhammadusmanfarooq/pakistan-solar-radiation-dataset
4. Youtube comments under videos with keyword solar in the title, from a catered list of channels.
5. Yahoo Finance. Time Series Data on Power Production Companies in Pakistan.

## Data -> Scripts Relation:
1. extract_pdf_data -> provides pdf and csvs for manual extraction of data
2. extract_kaggle -> both kaggle datasets
3. extract_stocks -> Yahoo Finance
4. extract_google -> Youtube comments
